Key Features

Everything you need to manage your finances effectively

Transaction Tracking

Record all your income and expense transactions with detailed categorization and date tracking.

Financial Reports

View summaries by category and month. Understand where your money goes with percentage breakdowns.

Secure Access

Your financial data is protected with user authentication. Only you can view and manage your transactions.

Balance Overview

See your current balance at a glance — the difference between total income and total expenses.

Category Management

Organize transactions into categories like Food, Bills, Transport, Payment, Investments and more.

Full CRUD Operations

Create, view, edit and delete transactions easily. All data is validated for correctness.

How it works

Register a free account and start tracking your finances immediately. Add transactions with a name, category, amount, type (income or expense), and date. The system automatically calculates your balance and generates detailed reports grouped by category and month.

Built with modern technologies

This application is built using Spring Boot framework with Spring Security for authentication, Spring Data JPA for database access, Thymeleaf for server-side rendering, and Bootstrap 5 for responsive design. All input data is validated both on the client side (JavaScript) and server side (Bean Validation).
